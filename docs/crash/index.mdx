# FCL启动器崩溃排查与修复

请访问我们的[完整崩溃排查指南](./troubleshooting.mdx)，了解如何诊断和修复常见的游戏崩溃问题。

## 常见崩溃类型

- **内存不足** - 当分配给Minecraft的内存不足时
- **模组冲突** - 当安装的模组之间存在兼容性问题时
- **游戏版本不兼容** - 当模组与游戏版本不匹配时
- **Java版本问题** - 当使用的Java版本不兼容时
- **渲染器错误** - 当渲染器配置不正确时

## 获取帮助

如果您遇到崩溃问题，请查看[详细排查指南](./troubleshooting.mdx)或联系我们的社区获取支持。
