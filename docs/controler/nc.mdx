# 新版控制器设置指南

FCL启动器的新版控制器提供了更加流畅和直观的操作体验，适合所有类型的游戏玩法。

## 主要特点

新版控制器相比传统控制器具有以下优势：

<ul style={{listStyleType: 'square', paddingLeft: '20px'}}>
  <li style={{margin: '8px 0'}}>更低的按键延迟</li>
  <li style={{margin: '8px 0'}}>支持多点触控</li>
  <li style={{margin: '8px 0'}}>可自定义的按键振动反馈</li>
  <li style={{margin: '8px 0'}}>支持手势控制</li>
</ul>

## 设置说明

### 启用新版控制器

<div style={{backgroundColor: '#f8f9fa', padding: '15px', borderRadius: '6px', marginBottom: '20px'}}>
  <ol style={{marginLeft: '20px', paddingLeft: '0'}}>
    <li>打开FCL启动器</li>
    <li>进入"设置"菜单</li>
    <li>选择"控制器"选项</li>
    <li>切换到"新版控制器"</li>
    <li>点击"应用"保存更改</li>
  </ol>
</div>

### 自定义按键布局

新版控制器支持高度自定义，您可以：

1. **调整按键位置**：长按任何按键并拖动到您喜欢的位置
2. **调整按键大小**：使用两指缩放手势改变按键大小
3. **更改按键透明度**：在按键编辑菜单中调整透明度滑块

<div style={{display: 'flex', justifyContent: 'center', margin: '20px 0'}}>
  <div style={{textAlign: 'center', maxWidth: '80%'}}>
    <p style={{fontStyle: 'italic', color: '#666'}}>提示：您可以创建多个控制器配置文件，针对不同的游戏类型快速切换</p>
  </div>
</div>
