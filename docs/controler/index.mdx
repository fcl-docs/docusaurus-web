# FCL启动器控制器设置

FCL启动器提供了多种控制器选项，让您可以获得最佳的游戏体验。请选择以下其中一个指南继续：

- [触控控制器](cc.mdx) - 经典触控控制器设置和自定义
- [新版控制器](nc.mdx) - 改进的控制器系统，提供更多功能和更好的响应性

无论您是休闲玩家还是竞技玩家，FCL启动器都能提供适合您需求的控制选项。

## 控制器类型对比

| 特性 | 触控控制器 | 新版控制器 |
|------|------------|------------|
| 响应速度 | 标准 | 更快 |
| 自定义程度 | 基础 | 高级 |
| 多点触控 | 支持 | 增强支持 |
| 手势支持 | 有限 | 全面 |
| 系统资源占用 | 低 | 中等 |

请根据您的设备性能和游戏需求选择合适的控制器类型。
